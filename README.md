# Line Follower Car using 5-Channel IR Sensor Module
Hardware Components used:
- Arduino UNO R3
- L289N H-Bridge Dual Motor Driver
- TCRT5000L 5-Channel Line Tracking IR Sensor Module
- x2 Geared DC Motors
- 12V 1800mAh Battery
